# Instrucciones para agentes

- Responde en español para cambios, resúmenes y PRs de este repositorio.
- Mantén el proyecto Android en Kotlin, Jetpack Compose, Material 3 y Gradle Kotlin DSL.
- No versiones `local.properties` ni archivos generados por Gradle/Android Studio.
- Antes de entregar cambios de código, ejecuta la validación Gradle más específica posible; para la app base usa `gradle assembleDebug`, ya que el wrapper JAR no se versiona.
- No implementes catálogo de opciones mpv, importación/exportación ni plantillas hasta que se solicite explícitamente.
